- Descrição do projeto -

Bulma é uma Ferramenta de Framework de Css. É modular e usa FlexBox, um modelo de layout unidimensional (lida com o layout em uma dimensão de cada vez - seja uma linha ou uma coluna) que é capaz de organizar espacialmente os elementos em uma interface, além de possuir capacidades avançadas de alinhamento.
